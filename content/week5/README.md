**Quick Sort**. 
>像二分法，分成左邊部分和右邊部分，把大問題分成小問題處理. 

大致上來說，快速排序法就是先在序列中找出一個元素作為支點(pivot)，然後想辦法將比支點的元素移動到支點元素的左邊，
比支點大的元素移動到支點元素的右邊，接著再用同樣的方法繼續對支點的左邊子陣列和右邊子陣列進行排序。


*適用於**資料量大**時。
