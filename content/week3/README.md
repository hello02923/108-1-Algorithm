## week 3
 > Topic : Stack and Queue
   * stack 疊盤，後進先出/像是復原（在儲存前都存在這)  
   #兩個指令  
    ＿push( )：新增元素至陣列的尾端，並回傳陣列的新長度。  
    ＿pop( )：刪除且回傳陣列的最後一個元素。  
    
   * queue 排隊不能插隊，先進先出/一次只能做一件事. 
   #兩個指令  
   ＿shift( )刪除且回傳陣列的第一個元素。  
   ＿push( )：新增元素至陣列的尾端，並回傳陣列的新長度。    
   
   
 stack and queue 都是一種儲存資料的方式
